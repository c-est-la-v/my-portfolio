Projects

KV62: Discovering Tutankhamon

Created for the Virtual Reality course 2023-2024 at Politecnico di Torino, this is a Desktop-VR application (with potential developments for headset) that allows users to explore the tomb of Tutankhamun, engaging in interactive and active learning through various features. The project was developed with Unity.
The app is available here.
Here you can find the italian version of the pitch.

Super Sonic

"SuperSonic: A Supersonic Social Media Campaign" has been created for the 2023-2024 Future Storytelling course at Politecnico di Torino. The aim was to provide a social media campaign for a Politecnico project funded by the European Union, called "More&Less" (which aims to study sustainable supersonic flight). The campaign was presented directly to the European Union funders. LinkedIn, TikTok, and X were used as platforms, and the data analysis was done with 4CAT. Here you can find the final report.

Chemical Chronicles

For the 2023-2024 Game Design course at Politecnico di Torino, a pitch for an entire video game (Chemical Chronicles: The Shrinking Adventure) was created (it is available here). Additionally, a prototype was developed using Unity, available here. It is a platformer set in a chemistry lab, playable on desktop with keyboard or controller.

ElemHands

ElemHands is a project developed by the student team "LevelUpLab". It is a VR video game, developed for Meta Quest 3, where simplified sign language is the key to solving puzzles. The preliminary pitch is available here. Unity was used, along with FMOD and the META XR plugin. The prototype was showcased in the Indie Dungeon area at Milan Games Week 2024.

Tale of Contour

This CGI Animated Spot, created by Milford Creative Studio, was partially recreated with Blender for the 2023-2024 Computer Animation course at the Politecnico di Torino. The aim was to utilize all the techniques learned during the course to produce a video at least 30 seconds long that would resemble the original advertisement. The result is available here.
